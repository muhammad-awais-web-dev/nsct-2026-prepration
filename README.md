# NSCT 2026 Preparatory Quiz Engine

An interactive, minimalist, and high-performance preparatory tool for the **National Skills Competency Test (NSCT) 2026**.

[![GitHub Stars](https://img.shields.io/github/stars/muhammad-awais-web-dev/nsct-2026-prepration?style=social)](https://github.com/muhammad-awais-web-dev/nsct-2026-prepration)
[![GitHub Followers](https://img.shields.io/github/followers/muhammad-awais-web-dev?style=social)](https://github.com/muhammad-awais-web-dev)

## 🚀 Overview

The NSCT 2026 is a critical assessment for computing graduates in Pakistan. This application provides a comprehensive environment to master the 10 core competency areas defined by the HEC and P@SHA.

### Key Features

- **1000+ Question Database:** Powered by AI to provide a vast and diverse set of questions.
- **Local Storage Integration:** Save AI-generated questions locally to build your own custom question bank.
- **Detailed Rationales:** Every answer (correct or incorrect) comes with a clear explanation of the underlying concept.
- **Minimalist Design:** A focused, monochrome interface with high-contrast feedback (Green/Red) to minimize cognitive load.
- **Syllabus Aligned:** Questions are weighted and categorized according to the official NSCT 2026 syllabus.
- **Real-time Feedback:** Instant validation and conceptual reinforcement.

## 🛠️ Tech Stack

- **Frontend:** React 19, TypeScript, Vite
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **AI Integration:** Google Gemini API (for dynamic question generation)
